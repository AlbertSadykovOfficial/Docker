

		Docker Hub

				Выгрузка проекта:

					Создаем псевдоним для образа в пространсве имен Docker Hub. <username>/<rep_name>
						
						docker tag identidock:latest albertsadykovofficial/identidock:0.1
						
					Выгрузка образа с использованем только что созданного псевдонима.
						
						docker push albertsadykovofficial/identidock:0.1


				После этого репозиторий можно загрузить командой docker pull.


				Для поледующих обновлений следует проделывать приведенные ранее операции.
				Если нужно обновлять образы при внемении иземенений в исходынй код, 
				то следует использовать автоматические сборки, реализованные на git
				через сервисы GitHub и Bitbucket.
