

 		Первые шаги

 			Чтобы создать контейнер следует выполнить команду:

 					docker run <command>

 					Примеры:

 						Напечатать текст в ОС:
 							docker run debian echo "Hello world"

 			После этой команды докер скачает с Docker Hub последнюю
 			версию образа (если она еще не скачана) и запустит его в контейнере.
 			Если запуск был первым, то это займет некоторое время,
 			последующие запуски не займут много времени.

 			Каждому запущенному контейнеру дается его некоторые параметры,
 			которые можно посмотреть командой (docker ps). Парамтры ID и NAMES
 			идентифицируют контейнер среди всех. Поле Name по умолчнанию 
 			задается из случайного прилагательного и Имени известного ученного.


 			Удалить контейнер:

 					docker rm <DOCKER_NAME>

 			Удалить все остановленные контейнеры и вывести их ID :

 					docker rm -v $(docker ps -aq -f status=exited)

			Подробности о всех контейнерах - запущенных и остановленных 
			(запустить в соседнем терминале от терминала докера):
					
					docker ps

			Больше информации о контейнере:
				
					docker inspect <DOCKER_NAME>
			
			Именование контейнеров (-h):

 					docker run -h CONTAINER -i -t debian /bin/bash

			Запустить оболочку bash debian в контейнере 
				docker run -i -t debian /bin/bash

				Флаги -i, -t создают сеанс интерактивной работы.

				Контейнеры работают пока работает их основной процесс,
				если мы запускаем какое-то приложение (у нас - bash),
				то при его закрытии закроется и сам контейнер.
				Поэтому, если мы введем: exit, то закроется 
				и bash, и контейнер.

			Посмотреть все действия, которые выполнялись в контейнере:

					docker logs <DOCKER_NAME>
