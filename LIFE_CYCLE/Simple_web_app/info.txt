

		Простое web-приложение

				для пользоватлей при вводе неоторого произвольного текста.

				помощи уникального изображения, сгенерированного по имени и IP.
				( У GitHub они генерируются, если не установлено изображение в профиле )
				Будет скомпановано приложение на базе микросервисов
