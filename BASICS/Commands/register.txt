

		Работа с реестрами

				Работа с реестрами, такими как Docker Hub.
				Аунтификация сохраняется в файле .dockercfg, расположенный в домашнем каталоге.

				docker login - регистрация или вход.
				docker logout - выход
				dicker pull - Загружает образ ИЗ реестра.
				docker push  - Выгружает образ в заданный реестр.
				docker search - Выводит список общедоступных репозитореев из реестра Docker Hub.
