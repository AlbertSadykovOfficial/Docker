

		Именвание образов


				При распространении образов следует использовать точные и содержательные
				имена и теги, которые присваиваются командой docker tag

					Опредилим имя репозитория и тега:
						
						docker build -t "identidock:0.1" .
					
					Установить соответствие имени c образом, который ссылается на имя
					пользователя albertsadykovofficial в Docker Hub
						
						docker tag "identidock:0.1" "albertsadykovofficial/identidock:0.1"


			Ограничения:

					Имя тега должно состоять из букв верхнего и нижнего регистров,
					и из символов "." и "-", но их запрещено использовать вначале.
