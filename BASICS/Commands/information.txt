

		Информация

				docker info - выводит информацию о системе Docker и хосте на котором она работает
				docker help  - информация по использлванию
				docker version - версия клиента Docker

		Информация о контейнере

				docker diff - информация о ФС контейнера по сравнению с ФС образа.
				docker events - выводит в реальном времени события от демона демону. (Выход: Ctrl + C)
				docker inspect - подробная ниформация о контейнерах и образах. (-f - для шаблона Языка - Go)

				docker logs - журналы
				docker port - список открытых портов.
				docker ps - Общая информация о работающих контейнерах.

				docker top - Информация о процссах, выполнящихся внутри контейнера.
